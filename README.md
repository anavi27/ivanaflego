## Design & Visual Identity
As someone who bridges the gap between design and code, the development process began with detailed visual identity planning:

* **Figma:** Created a complete UI/UX prototype, defining grid systems and typography.
* **Adobe Photoshop:** Image processing and optimization of graphical elements for the web.
* **User Experience:** Focus on clean lines, intuitive navigation, and loading speed.

## Tech Stack

This project was built using core web technologies to ensure maximum control over performance:

* **HTML5:** Semantically correct structure optimized for SEO.
* **CSS3:** Advanced layouts using **Flexbox** and **Grid**, featuring full responsiveness.
* **JavaScript (ES6+):** Logic for interactive elements, smooth scrolling, and dynamic animations.
