Smart Spender Web Apliakcija

Ova web aplikacija za praćenje osobnih financija izrađena je kao projektni dio diplomskog rada. Izrađena je koristeći HTML za strukturu, CSS za dizajn i izgled te JavaScript za logiku, a za bazu podataka i hosting korišten je Firebase (Firebase Authentication, Firebase Hosting & Firestore). UI i UX dizajn izrađeni su u Figmi. Klikni na ikonu i pogledaj stranicu!
